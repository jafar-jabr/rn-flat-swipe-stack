# rn-flat-swipe-stack

React native horizontally swipe stack

## Installation

```sh
npm install rn-flat-swipe-stack
```

## Usage

```js
import RnFlatSwipeStack from "rn-flat-swipe-stack";

// ...

const result = await RnFlatSwipeStack.multiply(3, 7);
```
